    Propósito de sitio: Por medio de este proyecto se pretende dar cumplimiento a la acividad propuesta aplicando los conocimientos adquiridos en la unidad 1 del curso Diseño de Sitios Web. Para esta actividad, el tema seleccionado es: Etiquetas Básicas de HTML Atributos y Valores
    Daremos una breve explicación de las principales Etiquetas utilizadas en "HTML" Debemos tener en cuenta que bajo la estructura "<>" se inicia la Etiqueta y con "" le damos cierre
    HTML:
    Esta etiqueta inicia y finaliza todo documento HTML y sirve para indicar que el texto se puede analizar
    Head
    Esta delimita el contenido de la cabecera en el documento, demarca el titulo de la página
    Title
    Aparece en la barra de titulos del navegador web
    b
    permite aplicar negrita al texto indicado
    <"!--">
    Esta línea de código le permite comentariar al programador y llevar un orden, adicional, si alguien más va a trabajar sobre el código, las líneas comentariadas le permitiran entender facilmente la estructura del código

    Formatos de Textos

    <"em">
    Aplica letra cursiva al texto indicado
    <"u">
    Aplica subrayado al texto indicado
    En este apartado describiremos las etiquetas más utilizadas junto con sus propiedades:
    Font
    Determina un texto con un formato determinado definido por sus propiedades
    Ejemplo:
  • Font size x
    • Determina el tamaño de la letra
  • Font color
    • Esta etiqueta permite seleccionar el color para el texto deseado, generalmente se escribe bajo código hexadecimal de la A a la F
  • Font face
    • Con esta etiqueta seleccionamos el tipo de fuente que se desea aplicar al texto
    Titulos
    h1, h2, h3, h4, h5, h6: Permiten caracterizar y tipificar titulos dentro del texto
