    Propósito de sitio: Por medio de este proyecto se pretende dar cumplimiento a la acividad propuesta aplicando los conocimientos adquiridos en la unidad 1 del curso Diseño de Sitios Web. Para esta actividad, el tema seleccionado es: Etiquetas Básicas de HTML Atributos y Valores
    Daremos una breve explicación de las principales Etiquetas utilizadas en "HTML" Debemos tener en cuenta que bajo la estructura "<>" se inicia la Etiqueta y con "" le damos cierre
    HTML:
    Esta etiqueta inicia y finaliza todo documento HTML y sirve para indicar que el texto se puede analizar
    <"Head">, <"/Head">
    Esta delimita el contenido de la cabecera en el documento, demarca el titulo de la página
    <"Title">, <"/Title">
    Aparece en la barra de titulos del navegador web
    <"b">, <"/b">
    permite aplicar negrita al texto indicado
    <"!--">
    Esta línea de código le permite comentariar al programador y llevar un orden, adicional, si alguien más va a trabajar sobre el código, las líneas comentariadas le permitiran entender facilmente la estructura del código

    Formatos de Textos

    <"em">
    Aplica letra cursiva al texto indicado
    <"u">
    Aplica subrayado al texto indicado
    En este apartado describiremos las etiquetas más utilizadas junto con sus propiedades:
    Font
    Determina un texto con un formato determinado definido por sus propiedades
    Ejemplo:
  • Font size x
    • Determina el tamaño de la letra
  • Font color
    • Esta etiqueta permite seleccionar el color para el texto deseado, generalmente se escribe bajo código hexadecimal de la A a la F
  • Font face
    • Con esta etiqueta seleccionamos el tipo de fuente que se desea aplicar al texto

    Titulos

    h1, h2, h3, h4, h5, h6: Permiten caracterizar y tipificar titulos dentro del texto, cada uno de estos tiene un cierre anteponiendo /

    Formato para párrafos

    <"br">
    Introduce un salto de línea, esta etiqueta se cierra así misma
    <"p">, <"/p">
    Delimita un párrafo de texto
    A continuación se muestran algunos parámetros para esta etiqueta
  • <"p align="center">
    • Aplica alineación centrada
  • <"p align="left">
    • Aplica alineación hacía la izquierda
  • <"p align="right">
    • Aplica alineación hacía la derecha
  • <"p align="justify">
    • Aplica alineación justificada
    <"hr">
    Inserta una línea horizontal
    <"div">
    Utilizado para delimitar una sección dentro de un documento, agrupando un número de elementos para agregar estilos seleccionados

    Creación de Listas

    Lista no numerada o no organizada:
  • <"ul">, <"/ul">
    • Inicia la Lista: <"li">, <"/li"> Lista numeradas u organizadas:
  • <"ol">, <"/ol">
    • Inicia la Lista: <"li">, <"/li">

    Inserción de Imágenes

  • <"img src=dirección de la imagen>:
    • Inserta la imagen de la dirección indicada
  • <"img border="X">:
    • Establece un borde a la imagen de la cantidad de pixeles indicados
  • <"img alt="texto explicativo">:
    • Se muestra un texto al pasar el mouse sobre la imagen
  • <"img align="bottom">:
    • Alinea la imagen al inferior del texto
  • <"img align="middle">:
    • Alinea la imagen al medio del texto
  • <"img align="top">:
    • Alinea la imagen al superior del texto
  • <"img align="left">:
    • Alinea la imagen hacía la izquierda del texto
  • <"img align="right">:
    • Alinea la imagen hacía la deerecho del texto
  • <"img hspace="x">:
    • Espacio horizontal en pixeles entre la imagen y el texto
  • <"img vspace="y">:
    • Espacio vertical en pixeles entre la imagen y el texto

    Inserción de Tablas

  • <"table">, <"/table">
    • Define donde inicia y termina la tabla
  • <"td">, <"/td">
    • Indica el comienzo y el fin de cada una de las columnas o celdas dentro de las filas
  • <"tr">, <"/tr">
    • Indica el comienzo y el fin de cada una de las filas de la tabla
  • <"table width="x">
    • Determina la anchura de la tabla en pixeles
  • <"table height="y">
    • Determina la altura de la tabla en pixeles
  • <"table border="x">
    • Establece el grosor en pixeles del borde de la tabla
  • <"table cellspacing="x">
    • Determina el espacio en pixeles entre las celdas
  • <"table cellspading="x">
    • Determina el espacio en pixeles entre el borde y el texto

    Creación de Enlaces

  • <"a href="www.google.com"">Nombre del enlace<"/a">
    • Al hacer click sobre el texto del enlace el vinculo creado en href abre
  • <"a href="mailto:ejemplo@ejemplo.com"">Nombre del enlace<"/a">
    • Al hacer click sobre el texto del enlace una ventana se abre para redactar un correo al mail indicado
